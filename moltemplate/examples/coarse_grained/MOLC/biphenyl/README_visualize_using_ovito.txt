1) Install OVITO, if not installed already  (https://www.ovito.org)

2) After you have run the simulation invoke ovito and supply the name of the
  trajectory (dump) file that you want to read:

ovito traj.lammpstrj

3) In the "External File" panel (bottom-right), scroll down until you see
    the checkbox named "File contains time series".  Click this box.

4) Click on the "User-defined mapping to particle properties" button.

5) Click on the "Edit column mapping..." button. (Scroll down if not visible.)

6) Change the settings so that they match those shown in this image:
   "README_visualize_OVITO_settings.png" file (in the parent directory).
